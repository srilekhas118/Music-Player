# Music-Player
# An interactive and responsive music player built with HTML, CSS, JavaScript, and React. This project showcases modern front-end development practices with a clean design and smooth user experience.
## Features
- Play, pause, and skip tracks  
- Volume and progress controls  
- Playlist support  
- Responsive design for desktop and mobile  
- Modular React components for scalability
