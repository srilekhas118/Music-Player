# 🎵 Music Player

Tech Haven Music Player is a modern, feature-rich web application designed to bring your music experience to life. Built with **React**, and **Firebase**, it offers seamless playback, album management, and user authentication — all wrapped in a clean, responsive interface.

---

## 🚀 Features

- 🎶 **Music Playback** — Stream and enjoy your favorite tracks with a built-in audio player.
- 📀 **Album Management** — Create, edit, and browse albums with ease.
- 🔐 **Authentication System** — Secure login, registration, and password reset functionality.
- 👤 **User Profiles** — Update personal details, manage accounts, and customize settings.
- 📊 **Admin Dashboard** — Manage albums, users, and content with powerful admin tools.
- 📱 **Responsive Design** — Optimized for both desktop and mobile devices.
